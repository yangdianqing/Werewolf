## About Werewolf

狼人杀讨论网站

- 基于自己认可的原则
- 讨论狼人杀游戏中的行为，分享行为的可行性
- 讨论狼人杀录像
- 给狼人杀选手打分
- 通过给对手的影响，反向作用于用户自己，让自己说话的分量、评价更有影响
- 学习、研究分享狼人杀战术
